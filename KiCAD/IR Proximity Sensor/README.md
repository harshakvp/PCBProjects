# IR Proximity Sensor

A compact IR-based proximity sensor designed using **KiCad**. This project demonstrates schematic design, PCB layout, routing, and 3D PCB visualization workflows.

## Getting Started

### Clone Repository

```bash
git clone https://github.com/harshakvp/PCBProjects.git
```

### Navigate to Project

```bash
cd PCBProjects/KiCAD/IR\ Proximity\ Sensor
```

### Open in KiCad

```text
IR Proximity Sensor.kicad_pro
```
